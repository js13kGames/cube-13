[PLAY on js13kgames!]()

[PLAY LAST VERSION on GitHub!]()

## Development
Install
```
yarn
```

Build release build
```
yarn build
```

Build final build with `roadroller`
```
yarn build-roadroller
```

Build debug build
```
yarn watch:ts
yarn watch:rollup
```
